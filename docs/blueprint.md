# **App Name**: EcoGov

## Core Features:

- Animated Stats Display: Display animated statistics on the landing page, such as 'Waste Treated' and 'Citizens Trained'.
- Interactive Training Modules: Provide interactive training modules for citizens, covering topics such as waste segregation and composting.
- Waste Reporting with Geolocation: Allow citizens to report waste issues by uploading photos and using geolocation for automatic address detection.
- Data Visualization Dashboard: Present key performance indicators (KPIs) related to waste management, with graphical summaries of segregation rates and waste generation trends.
- E-Commerce Utility Shop: An e-commerce portal where citizens and organizations can purchase necessary waste utility items.
- Facility Locator: Use map to visualize the Biomethanation, Recycling and Waste-to-energy (WTE) plants on the map.
- Violation tool: Leverage AI to identify and flag waste management violations from user-submitted reports, enabling swift administrative action.

## Style Guidelines:

- Primary color: Soft green (#90EE90) to represent sustainability and environmental consciousness.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Dark green (#006400) to highlight interactive elements and important information.
- Body and headline font: 'Inter', sans-serif, for a clean, modern and readable look.
- Use lucide-react icons throughout the application to represent various features and actions.
- Employ a grid-based layout for consistent alignment and spacing across all pages and components.
- Implement smooth animations and micro-interactions using Framer Motion to enhance the user experience.
