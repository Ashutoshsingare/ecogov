# EcoGov - National Waste Governance Platform

This is a minor project for a college course.

## Overview

EcoGov is a web application designed to serve as a national platform for waste management. It aims to connect citizens, municipalities, and waste management facilities to create a more efficient and transparent system for waste governance.

### Key Features:
*   **Citizen Training:** Educational modules on waste segregation and composting.
*   **Waste Reporting:** Allows users to report illegal dumping and other waste-related issues.
*   **Facility Locator:** A map to find nearby recycling centers and waste treatment plants.
*   **Dashboard:** Provides an overview of waste management statistics for officials.
*   **e-Shop:** A marketplace for sustainable products.

## Tech Stack

*   **Frontend:** Next.js, React, TypeScript
*   **Styling:** Tailwind CSS, ShadCN UI
*   **Generative AI:** Google Genkit
*   **Maps:** Google Maps API

This project demonstrates the use of modern web technologies to address real-world environmental challenges.
